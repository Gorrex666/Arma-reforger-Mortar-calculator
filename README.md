HOW TO USE: 

(Install autohotkey1)

select mortar type, shell, rings of the shell

input distance to target in the first input field

input elevation difference in the second input field (if the target is +100 mts the input must be -100, if the target is -100 mts the input must be 100)

it will display the required mills 


HOTKEYS

F12: Close script

END: Reload script

UP : window always on top

` : select first input field and delete content
